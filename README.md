# 🛡️ PhishGuard AI Security
An Advanced Threat Detection system powered by a Hybrid Intelligence Model (Machine Learning + Google Gemini API) to identify and flag phishing URLs.

🚀 **[Live Demo: Try PhishGuard AI Here](https://phishguard-ai.streamlit.app)**

📱 **[Download Mobile App (.apk)](https://github.com/sorifhossain960-jpg/PhishGuard-AI/releases/tag/v1.0.0)**

## 🌟 Overview
PhishGuard AI provides real-time URL scanning to protect users from malicious phishing attempts. By combining traditional machine learning algorithms with the deep analytical capabilities of **Gemini**, it offers a highly accurate and interactive threat detection experience.

## 🛠️ Tech Stack
- **Frontend & Deployment:** Streamlit
- **AI & Logic:** Google Gemini API, Scikit-learn
- **Data Processing:** Python, Pandas, NumPy
- **Dataset:** `phish.csv` 

## ✨ Key Features
- **Live Web Interface:** Easy-to-use console for users to paste URLs and initiate deep scans.
- **Hybrid Intelligence:** Uses both statistical models and Generative AI for better accuracy.
- **Real-Time Analysis:** Instant results indicating whether a link is safe or potentially malicious.
